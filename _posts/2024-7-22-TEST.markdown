# tip:

使用库的时候，临时加环境变量的方法：

```c
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:$(pwd)
```

编译库的命令(范例)：

```c
gcc  -fPIC  -shared  -o   libadd.so  add.c
```
